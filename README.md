# POKERTI 德扑人格测试 (SBTI Remake)

> MBTI算什么？来测测你到底是哪种德扑玩家。

一个基于开源娱乐性人格测试项目（原 SBTI 测试）二次开发的**德州扑克主题专属测试**。本测试将原始的 15 维度人格模型重新解构为德扑玩家的 5 大生态模型，帮助你精准定位自己在牌桌上的精神状态和打牌风格。

## 在线体验

👉 [点击开始测试](https://chenyu1025.github.io/POKERTI/)

## 特性

- 🃏 **14种德扑玩家人格** — 12种标准类型（如“深海大鲨鱼”、“狂暴疯狗”、“无情跟注站”） + 2种隐藏/兜底类型（如“赛博赌神”）
- 📊 **15个德扑专属维度** — 激进攻击、风险上头、算计读牌、牌桌社交、德扑生活五大模型
- 🎯 **曼哈顿距离匹配** — 基于15维向量的科学匹配算法，精准计算你的德扑纯度
- 🔪 **隐藏彩蛋** — 赛博赌神人格触发机制（当你选择借钱越级打高额局找刺激时...）
- 📱 **移动端优先** — 响应式设计，手机体验友好
- 🔧 **易于定制** — 数据与代码分离，改 JSON 即可创建你自己的测试

## 五大德扑模型解析

1. **激进攻击模型（A）**：A1 翻前入池率、A2 翻后开火欲、A3 诈唬成瘾度
2. **风险上头模型（R）**：R1 资金管理观、R2 情绪抗波动、R3 英雄抓鸡欲
3. **算计读牌模型（C）**：C1 赔率计算力、C2 对手读牌度、C3 理论实践度
4. **牌桌社交模型（S）**：S1 牌桌话痨度、S2 表情管理力、S3 嘲讽互动度
5. **德扑生活模型（L）**：L1 熬夜修仙度、L2 复盘学习欲、L3 装备发烧度

## 评分算法

1. **求和**：每维度2题分值相加（范围2-6）
2. **分级**：≤3 → L（低），4 → M（中），≥5 → H（高）
3. **向量化**：L=1, M=2, H=3，生成15维数值向量
4. **匹配**：计算用户向量与每种德扑人格类型的曼哈顿距离
5. **排名**：距离升序 → 精准命中降序 → 相似度降序
6. **特殊覆盖**：赛博赌神彩蛋 > 正常匹配 > 混沌发牌员兜底（<60%）
